학습 흐름 (4단계)

  1. 1단계 · 8×8 체스판 도전 — 직접 도미노를 놓으며 "언제 가능/불가능한가" 탐구
  2. 2단계 · 패턴 찾기 (6×6 & 4×4) — 같은 색 / 다른 색 제거 비교 실험
  3. 3단계 · 증명 힌트 — 해밀턴 순환 경로를 스스로 발견
  4. 4단계 · 증명 풀이 — 순환 경로를 이용한 완성된 증명

다음 페이지 이동 암호

  • 1단계 → 2단계: chess
  • 2단계 → 3단계: domino
  • 3단계 → 4단계: challenge

각 페이지 맨 아래 암호 입력창에 입력하면 다음 페이지로 이동합니다. 수업 진행 타이밍에 맞춰 암호를 공개하세요.

마스터키 (선생님 전용 단축키)

  • 2단계 · Shift+T — 타이머 5초로 점프
  • 2단계 · Shift+F — 타이머 즉시 만료
  • 2단계 · Shift+U — 모든 스테이지 잠금 해제
  • 3단계 · Shift+F — 1~6단계 전부 완료 상태로 점프

한글 입력 상태에서도 작동합니다. 입력창에 포커스가 있으면 무시됩니다.
